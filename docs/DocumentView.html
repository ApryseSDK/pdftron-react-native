Type: function
Optional.
Source: DocumentView.js line 257
Parameters:
Name Type Description
zoom double

the current zoom ratio of the document
Example
<DocumentView
  onZoomFinished = {(zoom) => {
    console.log('Current zoom ratio is', zoom);
  }}
/>
